# Курс по DevOps

### Используемый стек

- ubuntu, alpine
- git
- jdk
- maven
- docker
- docker-compose
- VSCode

### ДЗ

Необходимо создать Docker-compose.yml описывающий 2 сервиса.
- 1-ый сервис - сборочный (приложение собирается)
- 2-ой сервис - продовый (приложение запускается)
  - должен быть минимальным по размеру (примерно 100мб)
